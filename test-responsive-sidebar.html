Responsive Sidebar Test

This page tests the responsive behavior of the sidebar layout.

Resize your browser window to test different breakpoints.

Test Instructions

  1. Desktop (> 768px): Sidebar should be visible on the left
  2. Tablet (≤ 768px): Sidebar should collapse, toggle button should appear
  3. Mobile (≤ 576px): Sidebar should be full-width when opened
  4. Toggle Button: Click to open/close sidebar on mobile
  5. Overlay: Click overlay backdrop to close sidebar
  6. Escape Key: Press Escape to close sidebar

Expected Behavior

  • ✓ Sidebar visible by default on desktop
  • ✓ Sidebar hidden by default on tablet/mobile
  • ✓ Toggle button appears on tablet/mobile
  • ✓ Sidebar slides in from left when opened
  • ✓ Overlay backdrop appears when sidebar is open
  • ✓ Clicking overlay closes sidebar
  • ✓ Pressing Escape closes sidebar
  • ✓ Body scroll locked when sidebar is open on mobile
