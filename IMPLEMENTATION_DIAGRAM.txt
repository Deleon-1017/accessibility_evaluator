┌──────────────────────────────────────────────────────────────────────────────┐
│ FUNCTION HIERARCHY                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

showGuidelineDetails(guideline)
│
├─ generateExampleSections(guideline, safeId)
│  │
│  ├─ Check if guideline.examples exists
│  │  │
│  │  ├─ YES: Generate before/after cards
│  │  │  │
│  │  │  ├─ generateCodeSections(example.before, 'before', safeId)
│  │  │  │  │
│  │  │  │  ├─ syntaxHighlightCode(html, 'html')
│  │  │  │  │  └─ syntaxHighlightLine(line)
│  │  │  │  │
│  │  │  │  ├─ syntaxHighlightCode(css, 'css')
│  │  │  │  │  └─ syntaxHighlightCSSLine(line)
│  │  │  │  │
│  │  │  │  └─ syntaxHighlightCode(js, 'javascript')
│  │  │  │     └─ syntaxHighlightJSLine(line)
│  │  │  │
│  │  │  └─ generateCodeSections(example.after, 'after', safeId)
│  │  │     └─ (same as above)
│  │  │
│  │  └─ NO: Return "Examples coming soon" message
│  │
│  └─ Return HTML string
│
├─ setupCopyButtons()
│  │
│  ├─ Attach click handlers to all .copy-btn elements
│  │
│  ├─ On click:
│  │  ├─ Get code from target element
│  │  ├─ Try navigator.clipboard.writeText()
│  │  │  ├─ SUCCESS: showCopyConfirmation(button)
│  │  │  └─ FAIL: fallbackCopyMethod(codeBlock, button)
│  │  │
│  │  └─ Show "Copied!" feedback
│  │
│  └─ Return
│
└─ setupLazyLoading(guideline)
   │
   ├─ Create IntersectionObserver
   │
   ├─ Observe all .visual-output-container elements
   │
   ├─ When container enters viewport:
   │  │
   │  ├─ Get render target ID
   │  ├─ Determine if before or after
   │  ├─ Get example data
   │  │
   │  └─ renderExampleVisualOutput(id, html, css, js, guideline)
   │     │
   │     ├─ Create iframe element
   │     ├─ Build complete HTML document
   │     ├─ Write to iframe
   │     ├─ Handle errors
   │     └─ Set timeout for long scripts
   │
   └─ Return
