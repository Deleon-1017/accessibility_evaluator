INTERACTION FLOW
================================================================================

USER JOURNEY:

1. User opens WCAG guideline modal
   │
   ├─→ Sees "Before and After Visual Presentation" section
   │
   └─→ Sees three view mode buttons at top

2. User clicks view mode button
   │
   ├─→ Side by Side: Shows both cards
   │
   ├─→ Slider: Shows interactive slider
   │
   └─→ Toggle: Shows switch button

3. User interacts with selected mode
   │
   ├─→ Side by Side: Scrolls to compare
   │
   ├─→ Slider: Drags to reveal differences
   │
   └─→ Toggle: Clicks to switch or enables auto-play

4. User learns about accessibility
   │
   └─→ Understands the impact and implementation
