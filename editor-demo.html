Code Editor with Issue Highlighting Demo

This demonstrates the new CodeMirror-based editor with line number tracking for accessibility issues.

Features

  • Line Numbers: Every line is numbered for easy reference
  • Issue Markers: Lines with issues show colored icons in the gutter
  • Color-Coded Issues:
    • ● Red (Error) - Critical accessibility issues
    • ⚠ Yellow (Warning) - Potential accessibility issues
    • ⓘ Blue (Info) - Accessibility suggestions
  • Syntax Highlighting: HTML code with proper formatting
  • Line Wrapping: Long lines wrap automatically

How to Use in Your Application

  1. Include CodeMirror libraries and code-editor-utils.js
  2. Initialize the editor in a div container
  3. After scanning HTML, call highlightIssues(editor, issues) with your results
  4. Issues array format: [{line: 5, severity: 'error', message: 'Missing alt attribute'}]
  5. Users can see exactly which lines have problems with hover tooltips
